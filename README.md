# MathOnTape

Signal flow as compositional architecture. Experimental electronic music where visual language mirrors sonic texture.

**Full case study:** [petersalvato.com/practice/mathontape/](https://petersalvato.com/practice/mathontape/)

## About

Dirty analog synths, magnetic tape, feedback loops deliberately routed. MICR typography, archival naming, halftone-as-tape-hiss. The visual identity and the sonic identity are one system with two surfaces.
